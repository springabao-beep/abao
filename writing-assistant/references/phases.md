# 工作阶段

---

## 阶段一：素材整理与灵感激发

### 六层采集框架

从以下六个层面引导用户收集素材。注意像聊天一样自然流动，不逐条填空；用户在某一层谈得深，就顺着挖下去。

**第一层：历史叙事**
- 目的地经历过哪些重大事件？官方叙事和民间说法一致吗？
- 追问："这个历史事件距离今天多少年？你能感知这个时间跨度吗？——如果感知不到，反而值得写。"

**第二层：地缘政治**
- 地理位置赋予了这个地方什么命运？邻国是谁？当地人怎么谈论"外面的世界"？
- 追问："这个地方的地形你看一眼地图就能理解它的命运吗？还是恰恰相反——位置很好但历史很惨？"

**第三层：权力与国家**
- 这里的政府大楼什么样？好进吗？当地人谈论政治用什么语气？
- 追问："如果你在这里做一件在国内习以为常的事，会发生什么？"

**第四层：经济**
- 街上什么店铺最多？什么在关门？物价如何？年轻人做什么工作？
- 追问："你注意到的'隐形经济'有哪些？为什么这些生意会存在？"

**第五层：社会结构**
- 你在这条街上能看到几种不同"阶层"的人？老人和年轻人的眼神一样吗？
- 追问："当地人焦虑什么？希望什么？——不是问出来的，是看出来的。"

**第六层：街头观察**
- 选三个感官细节：光线的颜色、空气的味道、背景音——共同营造了什么氛围？
- 追问："你注意到什么'反常'的东西？这种并置往往是好文章的入口。"

### 挖掘模式（模式 D）—— 素材少时用

用户说"素材很少"或"帮我整理素材"时，AI 切换为此模式。

**第一步：基础输入**
用户提供：目的地名称、照片描述（0~N 张）、零散笔记（可选）

**第二步：AI 并行启动两条线**

**研究线** — 按六层框架做深度搜索，每层 3-5 条关键事实 + 来源标注 + 追问钩子（AI 判断这条信息为什么可能跟用户有关）。不一次全抛，按追问节奏分阶段喂。

**照片线** — 对每张照片锚点式追问（不要求每张走完四层）：
1. 事实提取：这张在哪儿拍的？什么时候？
2. 多感官还原：当时的光线、气味、声音？
3. 反常挖掘：画面里有什么让你意外的人/物/并置？
4. 与研究碰撞：照片内容与背景知识之间的张力

**第三步：交叉追问**

AI 每轮自觉切换追问来源：研究材料中有没有和某张照片可连接的点？某张照片值得再挖一层？照片线索是否有研究材料可以放大？研究的发现没有被任何照片覆盖？——追问"你注意到……没有？"

**第四步：输出产出**

```
📋 可用素材清单（来源：照片/研究/回忆，不硬套六层归类）

1. [素材标题] → 一句话素材描述
   ✏️ 潜在写作方向：一句话建议

⚠️ 张力警报
——你的[碎片]与查到[事实]之间有张力，值得关注
```

### 跨层思考工具

| 技术 | 做法 | 示例 |
|------|------|------|
| **追问"不证自明"** | 当用户说一个"当然是这样"的判断时，追问背后前提 | "你刚才说的 X 很有意思。展开讲讲？" |
| **发现平行线** | 两个现象正在走相似的路，把它们并置 | "你说的 A 和 B 之间，有没有共享的模式？" |
| **逆推成因** | 从今天的样子倒推它为什么变成这样 | "你说今天这里是这样的——十年前什么样？什么驱动的变化？" |
| **找到核心意象** | 提炼一个物理事物承载全文意义 | "你全部素材里，有没有一个东西可以同时代表'此地'和'此地的历史'？" |

### 素材研究模式

当用户需要补充知识背景时，AI 主动搜索并按六层体系整理资料，与用户个人素材区分呈现。输出格式：

```
## 研究摘要：[目的地]

### 第一层：历史叙事
- [关键事实]（[来源网站标题](URL)）
- 📌 值得关注：[提示叙事张力或独特角度]
```

规则：
- **每条信息必须附带可点击的原文链接**，格式为 `[网站名称](URL)`，放在关键事实后面
- 区别 AI 资料与个人素材、不编造、优先本地视角、用户个人素材优先
- 如果来自多页搜索结果，优先引用信息最完整、来源最可靠的那一页

## 阶段二：架构搭建

### 四种结构

| 结构 | 线索 | 适合场景 |
|------|------|---------|
| **A：空间叙事型** | 地理移动（引子→起点→途中 encounters→终点回望） | 完整走访（古城、街区） |
| **B：主题聚焦型** | 核心主题贯穿，多重印证 | 有明确思想诉求 |
| **C：时间层累型** | 现在→过去→更早→带着历史眼光回到现在 | 历史底蕴深厚的目的地 |
| **D：人物驱动型** | 人物出场→故事→与地方的关系→回响 | 人文访谈类 |

**选择建议：**
- 有清晰行走路线 → A
- 有反复出现的关键词 → B
- 历史素材特别丰富 → C
- 有人物故事贯穿 → D
- 都不明显 → A（最安全）
- 多强并存 → A+C 或 B+C

选定结构后产出大纲（含章节核心内容与预期字数），不需要完美，写作过程中可偏离。

## 阶段三：写作执行

> **初稿的唯一目标是存在，而不是完美。** 不要在写作中反复修改同一段。

### 开头四模式

1. **反常瞬间** — 从一个打破预期的画面切入，制造悬念
2. **感官锚点** — 用一个强烈的感官细节将读者拉入场景（温度、气味、声音、光线）
3. **历史纵深** — 第一句将读者拉到足够深的时间尺度
4. **人物登场** — 用一个人的话或一个动作直接展开

写完后自检：这个开头让我自己好奇了吗？有没有具体画面？是不是太概括？

### 段落节奏：三种力量的交替

| 力量 | 功能 | 写法 |
|------|------|------|
| **描写** | 让读者"看见" | 感官细节，少用形容词，多用名词和动词 |
| **叙事** | 让读者"跟随" | 时间线推进（我去了→我看见→我遇见） |
| **反思** | 让读者"思考" | 从画面升华，但篇幅不超过描写+叙事的 1/3 |

### 展示，而非告知

| 告知 | 展示 |
|------|------|
| 这个城市很穷 | 街边店铺大半关着门，开着的卖五块钱一碗的面 |
| 当地人很热情 | 问路时大爷直接拉着我走了两条街 |
| 这里历史悠久 | 地面上随便一个土堆，可能就是某个诸侯的陵墓 |

写完每句描述后检查：有没有对应的具体画面？没有的话，问自己"我为什么这么觉得"。

### 历史信息的融入三原则

1. **不要单独成段** — 历史像盐，融入菜里。每段历史信息必须连接到一个具体场景
2. **一个场景只配一个事实** — 读者的认知带宽有限
3. **历史需要你的态度** — 你的感受（困惑、震撼、疏离）才是文章的灵魂

### 人物对话处理

选取最核心的一两句对话，其余用旁白概括。不要变成"录音整理"。

**人物出场前自问：** 这个人为什么出现在我的游记里？故事揭示了此地的什么特质？和我的张力是什么？

### 叙事声音

| 类型 | 适合场景 |
|------|---------|
| **在场型** — "我"全程在场 | 空间叙事、街头观察 |
| **观察型** — "我"保持距离 | 人物驱动、人文访谈 |
| **反思型** — "我"退后一步 | 主题聚焦、历史反思 |

可在一篇文章中切换，但要有章法。

## 阶段四：修改与润色

初稿写完后冷却至少 24 小时（如有条件），然后分四层修改。

### 第一层：结构（最重要的）
- 开头是否有效？你想继续读吗？
- 每一段是否都有"存在理由"？没有就删
- 结尾是否回应了开头？——不是总结，是让开头的线在更高处收束

### 第二层：段落与句子
- 每段的展示/告知比例合理吗？有具体画面吗？
- 首句和末句有力吗？
- 句子长度有变化吗？
- 连接词（而/但是/然而/不过）是否重复出现？

### 第三层：措辞与韵律（时间紧张可跳过）
- 动词升级：走着→踱着/晃着/穿过；看着→凝视/瞥见/打量
- 抽象名词具象化："城市在衰败"→"商户比顾客多"
- 朗读测试：读一段出来，卡顿的地方就是需要改的地方

### 第四层：事实与合规检查

AI 须逐项核对用户文稿中的以下内容：

**事实核查（按优先级）：**
- 专有名词（人名、地名、寺名、藏品名）是否与权威来源一致
- 历史年代、建制沿革、事件描述是否有出入
- 引文、歌词、碑文、题记是否与原文逐字核对，不得凭记忆改写
- 建筑细节（结构名称、开间进深、材料等）是否准确
- 数据（游客数、年份、尺寸等）是否可验证

**敏感信息审查：**
- 涉及民族、宗教话题用词是否准确、中性（例如"入侵"应加引号或用"入主""进入"等更中性的表述）
- 涉及审查、下架、争议内容时避免直接断言，改用含蓄、个人化的表达
- 不直接透露或暗示违反中国法律法规的行为细节
- 不编造来源不明的政治历史判断

**原则：** 事实错误必须修正使与权威来源一致，敏感措辞调整为合规表达。两种修改都应在反馈中指出原问题并说明修改理由，帮助用户理解为什么需要调整。

## 附录：常见误区

1. **写得太"用力"** — 每句都想出彩。解法：初稿平淡，修改只保留自然出彩的
2. **站得太远** — 像学术报告，没有"我"的在场感。解法：每段素材后加一句"我当时的感觉"
3. **"著名的"陷阱** — 每个地方都加"著名的……"。解法：你只需要说出它在你面前的真实样子
4. **面面俱到** — 试图一次写全所有观察。解法：选一个核心意象做减法
5. **过早开始** — 素材没整理完就动笔。解法：好的素材整理完成时，你会感觉到文章"自己想被写出来"
