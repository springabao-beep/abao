# WorkBuddy Skills

WorkBuddy 自定义技能集合，用于提升 AI 助手的工作效率。

## 技能列表

| 技能 | 说明 |
|------|------|
| **daily-log-report** | 日常工作日志系统与飞书日报推送 |
| **email-writer** | 英文邮件写作，外企商务风格 |
| **idp-excel-analyzer** | 个人发展计划表（IDP）Excel 四维度分析 |
| **meeting-minutes-generator** | 会议逐字稿转结构化周会纪要 |
| **ppt-template-generator** | 品牌模板 PPT 生成 |

## 使用方式

在 WorkBuddy 对话中直接输入相关指令即可触发对应技能。
