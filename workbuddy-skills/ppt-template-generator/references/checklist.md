# Career Compass Format Checklist

Check before exporting a PPT.

- [ ] All titles use `#9D3D1D` on light background or white on dark
- [ ] Fonts: Futura Bk BT or Arial — no Times New Roman, Courier, Wingdings
- [ ] No font size smaller than 9pt
- [ ] Slide dimensions: 16:9 (13.33" × 7.50")
- [ ] Slide numbers: bottom-right, 11pt
- [ ] Colors: only Career Compass palette + white + black
- [ ] No text overflow / cut-off (use word_wrap=True)
- [ ] Consistent margin: 0.5" on all sides
